使用步骤

1

下载安装包

点击下载按钮获取 .exe 安装包，右键选择以管理员身份运行。

2

一键运行

自动检测并安装缺失组件，全程无需输入任何命令。

3

选购服务

选择一家 API 服务商，点下一步跳转官网，注册选购合适套餐后生成 API key ，复制备用。推荐使用 DragonCode，稳定不掺水。

4

配置模型

填入服务商提供的 API Key，测试连接成功后点击完成，安装完毕。（API Key 仅保存在您的系统配置中，本软件不读取、不存储。）

5

右键启动

在任意文件夹内右键，选择「从这里启动 Claude Code」，即可开始您的 AI 旅程。
